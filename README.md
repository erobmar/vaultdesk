# VaultDesk Alpha 1.0

Aplicación de escritorio para gestión local y cifrada de credenciales.

## Estado
Versión funcional 1.0

## Ejecución
No requiere instalar Java si se usa el paquete nativo incluido en la release.

## Aviso
Aplicación de ámbito académico. No se recomienda su uso para operaciones críticas.

## Plataformas
- Windows: app-image portable
- Linux: app-image / tar.gz
- macOS: app-image experimental
